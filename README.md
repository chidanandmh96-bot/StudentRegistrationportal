# Student Registration Portal

## 📌 Project Overview
This is a Spring Boot CRUD application for managing student registration details. It allows users to Create, Read, Update, and Delete student records.

## 🚀 Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL / PostgreSQL
- Maven
- REST API
- Postman

## 📂 Features
- Add new student
- View all students
- View student by ID
- Update student details
- Delete student

## 📌 REST APIs

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /students | Add Student |
| GET | /students | Get All Students |
| GET | /students/{id} | Get Student By ID |
| PUT | /students/{id} | Update Student |
| DELETE | /students/{id} | Delete Student |

## ▶ How to Run
1. Import project in STS / Eclipse
2. Configure database in application.properties
3. Run SpringBootApplication
4. Test APIs using Postman
